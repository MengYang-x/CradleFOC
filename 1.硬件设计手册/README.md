## 1.1 极对数判断

- 极对数：看转子上的磁铁个数，一般为2的倍数，极对数 = 转子磁铁个数 / 2。
- 使用直流电源进行测量：将电源电流限制到额定电流的10%左右，然后接电机的任意两相。用手抖动转子一周，有几次卡顿，极对数等于转动过程中卡顿的次数。

## 1.2 相电阻测量

使用万用表分别测量电机三相中的任意两相，然后取平均值，得到的为线电阻，之后需要再除以2即为相电阻值。

线电阻 = 2 * 相电阻
